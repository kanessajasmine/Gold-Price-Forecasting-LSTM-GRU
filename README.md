# Gold-Price-Forecasting-LSTM-GRU
Gold price forecasting using LSTM and GRU deep learning models.

## Overview
This project develops deep learning models using LSTM and GRU architectures to forecast gold prices based on historical data and economic indicators.

## Technologies
- Python
- TensorFlow
- Pandas
- NumPy
- Matplotlib

## Workflow
1. Data Collection
2. Data Cleaning
3. Feature Engineering
4. LSTM Training
5. GRU Training
6. Model Evaluation

## Evaluation Metrics
- RMSE
- MAE
- R²
