# Gold Price Forecasting Using Hybrid LSTM-GRU

## Overview
This project develops a hybrid deep learning model combining Long Short-Term Memory (LSTM) and Gated Recurrent Unit (GRU) to forecast gold prices in Indonesia. Unlike conventional forecasting approaches, this study incorporates multiple macroeconomic indicators including inflation, interest rates, exchange rates, and the US Dollar Index to improve prediction performance.

## Objectives
- Forecast future gold prices using deep learning techniques.
- Analyze the impact of macroeconomic indicators on gold price movements.
- Compare forecasting performance using machine learning and deep learning approaches.
- Generate insights for data-driven financial analysis.

## Dataset
The model utilizes historical gold prices and economic indicators from various public sources:
- Gold Price (IDR)
- Gold Futures
- Inflation Rate
- Bank Indonesia Interest Rate
- Federal Reserve Interest Rate
- USD/IDR Exchange Rate
- US Dollar Index (DXY)
