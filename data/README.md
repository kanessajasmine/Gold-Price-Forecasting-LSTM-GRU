# Dataset Description
This folder contains datasets used for gold price forecasting.

## Dataset List
| File | Description |
|--------|-------------|
| Harga Emas IDR.csv | Historical gold prices in Indonesian Rupiah |
| Data Historis Emas Berjangka.csv | Historical gold futures prices |
| Data Historis USD_IDR.csv | Historical USD to IDR exchange rates |
| Data Historis US Dollar Index.csv | Historical US Dollar Index (DXY) |
| Data Inflasi.xlsx | Indonesian inflation rate data |
| suku bunga.xlsx | Bank Indonesia interest rate data |
| suku bunga the fed.csv | Federal Reserve interest rate data |
| final_data.csv | Integrated dataset after preprocessing and feature engineering |

## Data Sources
- Bank Indonesia
- Investing.com
- Public economic and financial datasets

## Notes
The final_data.csv dataset was generated after data cleaning, integration, normalization, and feature engineering processes.
